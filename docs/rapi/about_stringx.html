about_stringx: Drop-in replacements for base R string functions powered by stringi¶

Description¶

stringx reimplements the built-in R functions for string processing based on stringi – a mature R package for fast, correct, consistent, and convenient text manipulation. Thanks to the ICU library, we obtain predictable results on every platform, in each locale, and under any native character encoding.

Keywords: R, text processing, character strings, internationalisation, localisation, ICU, ICU4C, i18n, l10n, Unicode.
